# dotfiles

> Dotfiles for vim, zsh, tmux, atom and some other stuff

![image of uxterm](dotfiles.png)

## About

This is my dotfiles. Handled using [gnu stow](https://www.gnu.org/software/stow/).

**Includes**:

- vim
- tmux
- zsh
- misc (xorg ++)
- atom
- arch

## Install

```bash
$ make <whatyouwant>
```

_Example_: Install vim, zsh & tmux config

```bash
$ make vim zsh tmux
```
